## 登录验证
### 1.发送code换取sessionKey

接口地址：/wechat/hxcy/auth/wxlogin

请求方式：POST

请求参数：


| 参数名 | 数据类型 | 备注 | 必传 |
| ------|----------|------|------|
| code| String|微信登录code| y|

返回参数：

| 参数名 | 数据类型 | 备注 |
| ------|----------|------|
| token| String| token存放在本地，过期需要重新调用该接口获取|
